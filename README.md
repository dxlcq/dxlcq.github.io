个人网站

**如何使用**

1. 安装 `python`

    ```shell
    sudo apt install python3 python3-pip
    ```

2. 安装 `mkdocs` 及 `mkdocs-material`

    ```shell
    pip install mkdocs mkdocs-material
    ```

3. 拉取本项目

    ```shell
    git clone https://github.com/dxlcq/dxlcq.github.io.git
    ```

4. 生成

    ```shell
    python -m mkdocs build
    ```
